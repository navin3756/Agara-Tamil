Agara Tamil Learning Privacy Policy

Effective date: June 8, 2026

Agara Tamil Learning is an educational Tamil learning app for families and classrooms. This policy explains what information the app uses and how it is handled.

Android App

The Android app runs in local mode. It does not require account sign-in, does not show ads, and does not include in-app purchases in the current release.

  • Lesson progress and practice state may be stored locally on the device.
  • Microphone access is used only when a learner chooses local recitation recording practice. The recording stays on the device and is not uploaded by Agara.
  • The app uses internet access for the web-based app shell and linked learning resources.

Web App

The web version may offer optional Google sign-in for progress features. If a user signs in on the web, authentication and saved progress may be handled through Firebase services. Firebase configuration values in the web app are public client configuration, not private API secrets.

Children

Agara is intended for family-guided Tamil learning. Parents, guardians, or teachers should supervise children while using online resources linked from the app.

Data Sharing

Agara does not sell personal information. The current Android release does not intentionally share user data with third parties.

Contact

For privacy or support questions, visit Agara Support.
